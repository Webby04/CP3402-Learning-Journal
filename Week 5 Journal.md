### Content Insights
I learned that child themes are a safe way to customise a WordPress site because they inherit functionality from the parent theme while allowing changes to styling and layout. I also learned that child themes are useful because they allow custom changes to be made safely without affecting the original parent theme. This makes it easier to experiment with design changes while keeping the site stable.

### Career/Employability/Learning Insights
This week showed me how websites are often customised using existing themes rather than building everything from scratch. It also helped me understand the importance of testing changes before applying them permanently, especially when working with live sites. Learning how child themes work gave me a better understanding of how WordPress sites are structured in practice and how changes can be made without affecting the original theme.
